# NYDev Admin Panel

## Overview
Enterprise admin console for NYDev with RBAC enforcement, CMS controls, and project/team management.

## Setup
1. Install dependencies
   - npm install
2. Run locally
   - npm run dev

## Environment Variables
Create a .env file in Admin/:

VITE_API_URL=http://localhost:5000

## Scripts
- npm run dev
- npm run build
- npm run preview

## Docker
Build and run via the root docker-compose.yml
